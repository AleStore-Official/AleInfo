AleStore Official User Guide

AleLogin

  1. Open the AleLogin page.
  2. Select email login or a supported provider.
  3. Enter your credentials in the fields.
  4. Press the login button to continue.
  5. Confirm your email if required.
  6. After logging in, you are redirected to the main area.

AleRegister

  1. Open the AleRegister page.
  2. Fill in the required information.
  3. Accept the policies if requested.
  4. Submit the form to create your account.
  5. Check your inbox for the confirmation email.
  6. Confirm your email and then log in through AleLogin.

AleStore

  1. Open the AleStore page after logging in.
  2. Browse the available items and categories.
  3. Select an item to view its details.
  4. Use the available buttons to open or interact with the content.
  5. Return to other areas using the navigation options.

AlePortal

  1. Open the AlePortal page.
  2. Explore the icons representing apps and tools.
  3. Select an icon to open the corresponding app.
  4. Use the app directly inside the page.
  5. Return to the main portal when finished.

AleCrypt

  1. Open the AleCrypt page.
  2. Type your message in the text field.
  3. Press the button to generate the protected message.
  4. Copy the generated message and share it with the recipient.
  5. Paste a received message into the field to read it.

AleDJ

  1. Open the AleDJ page.
  2. Load an audio track using the controls.
  3. Use play and pause to control playback.
  4. Adjust the available controls to manage the audio.
  5. Stop playback when finished.

AleCassier

  1. Open the AleCassier page when required by an operation.
  2. Review the information displayed.
  3. Follow the steps shown to complete the action.
  4. Wait for the confirmation message at the end.

AleConfirm

  1. Open the AleConfirm page when redirected after an action.
  2. Read the confirmation message displayed.
  3. Use the available button or link to continue.
  4. Close the page when finished.

AleInfo

  1. Open the AleInfo page.
  2. Read the information about the ecosystem.
  3. Use the links to navigate to other services.
  4. Return here whenever you need an overview.
